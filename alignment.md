# Alignment Document (정렬 문서)

## Core Objective
- 웹 브라우저에서 렌더링되는 3D 방식의 프리미엄 검은색 육면체(Cube) 생성

## Confirmed Specs
- **Shape**: 6면체 (Cube)
- **Color**: Deep Black (조명을 받아 윤곽이 드러남)
- **Environment**: Bright Sky / Sunlight Theme
- **Rendering**: Three.js 기반 WebGL 가속 사용
- **Interaction**: 정지 상태
- **Lighting**: DirectionalLight (태양) + AmbientLight (환경광)

## Technical Constraints
- **Library**: Three.js (CDN)
- **Styling**: Vanilla CSS (Premium Dark Mode Aesthetics)
- **Logic**: Javascript (ES6+)

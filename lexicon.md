# Lexicon Document (용어 사전)

## Terminology Mapping
- **6면체 박스 (Cube)**: 6개의 면을 가진 입체 기하학 객체 (Three.js `BoxGeometry`).
- **3D 방식 (3D Rendering)**: WebGL을 통한 하드웨어 가속 렌더링.
- **검은색 (Black)**: 단순한 무채색이 아닌, 조명의 영향을 받는 물리 기반 재질(`MeshStandardMaterial`)의 검은색.

## Naming Conventions
- **Scene**: 3D 공간을 담는 최상위 컨테이너.
- **Cube**: 렌더링 대상이 되는 육면체 객체.
- **Renderer**: 브라우저 화면에 3D 결과를 그리는 엔진.

## UI Tone & Manner
- **Background**: 미니멀하고 세련된 다크 그라데이션.
- **Animation**: 끊김 없는 60FPS의 부드러운 회전.
